
This lib (secret-code-crc) provides you the ability to setup one or more "cheat code" style easter egg and prevent them from being decrypted.
You can use it on your web pages, or applications based on JavaScript.
You should NOT use it for security purpose.

Everything you need to know is inside the source files.
Try maker.hta if you use Windows, or change the EXT to .htm if you use other OS.
Further detail is needless if you know JavaScript.

To view a live example, please go http://sc2.178.com/info/ and type "operationcwal" in the void.
