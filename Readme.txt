
This lib (secret-code-crc) provides you the ability to setup one or more easter egg secret code and prevent from decrypt.
You can use it on your web pages, or applications based on javascript.
You should NOT use it in your security system.

Everything you need to know is inside the source files.
Try the maker-crc.hta if you use Windows, or change the EXT to .htm if you use other OS.
You won't need any more details if you know what's JavaScript.

To watch a live example, please go http://sc2.178.com/info/ and type "operationcwal" in the void.
